مشروع نظام الويب الاحترافي:
وصف المشروع
هذا المشروع هو عبارة عن منصة ويب متكاملة مبنية باستخدام إطار العمل لارفل  يحتوي المشروع على لوحة تحكم، نظام مسارات مبرمج، وهيكلية احترافية لإدارة البيانات وتنسيق صفحات الموقع
التقنيات المستخدمة
* **Backend Framework:** Laravel (PHP)
* **Frontend:** HTML5, CSS3, Bootstrap
* **Database:** MySQL
* **Version Control:** Git & GitHub

أقسام المشروع الرئيسيسة
* **Navbar:** شريط تنقل علوي متجاوب للانتقال بين صفحات النظام.
* **Main Dashboard:** الواجهة الرئيسية لعرض محتويات وجداول البيانات.
* **Footer:** تذييل الصفحة يحتوي على حقوق النشر وروابط التواصل.
